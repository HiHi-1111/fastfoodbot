# Product Overview

## Project Purpose
Fast Food Bot for Bloxburg (Roblox) - An experimental automation script that plays the "fast food worker" role in the Roblox game Bloxburg using computer vision and OCR technology.

## Value Proposition
- Automates repetitive fast food order fulfillment in Bloxburg
- Uses advanced computer vision for real-time game state detection
- Provides educational demonstration of automation techniques
- Eliminates manual clicking and ingredient selection

## Key Features

### Computer Vision & OCR
- Screenshot capture and analysis of game window
- Template matching for ingredient and UI element detection
- OCR text recognition using Tesseract and EasyOCR
- Color analysis for game state identification

### Automation Capabilities
- Automated mouse control with native ctypes backend
- Order parsing and ingredient identification
- Multi-phase order processing (burgers, fries, drinks)
- Real-time GUI display of bot state and processing

### Configuration & Customization
- JSON-based button location configuration
- Configurable dialog regions for OCR
- Template image management for ingredient matching
- Screen resolution adaptability

## Target Users
- Game automation enthusiasts
- Computer vision learning projects
- Roblox Bloxburg players interested in automation
- Educational demonstrations of OCR and image processing

## Use Cases
- Automated fast food order fulfillment in Bloxburg
- Computer vision technique demonstration
- Educational automation project
- Game bot development learning
